<h1><b>Excel-Based Question Answering System using FAISS & Hugging Face LLM</b></h1>

<h2>📜 Overview</h2>

<h4>This project is a Retrieval-Augmented Generation (RAG) model that enables question answering over Excel data. It utilizes FAISS (Facebook AI Similarity Search) for efficient text retrieval and a Large Language Model (LLM) from Hugging Face for generating responses.</h4>

<h3>🚀 Features</h3>

<h4>✅ Query Excel Data Using Natural Language
  <br><br>
  📊✅ Fast Data Retrieval Using FAISS
   <br><br>
  ⚡✅ Context-Aware Answers from LLM 
   <br><br>
  🤖✅ No Hallucination - Answers Grounded in Provided Data
   <br><br>
  ✅✅ Useful for Enterprise & Business Data Analysis</h4>
   <br><br>


<h2>How It Works : </h2>

<h4>1️⃣ Load & Process Excel Data: Reads .xlsx files and converts them into text format.
   <br><br>
  2️⃣ Create & Store Embeddings: Converts text chunks into numerical embeddings and stores them in FAISS.
   <br><br>
  3️⃣ Set Up a Hugging Face LLM: Uses Mistral-7B-Instruct via the Hugging Face API.
   <br><br>
  4️⃣ Retrieve & Answer Questions: Retrieves relevant text using FAISS and passes it to the LLM for response generation.</h4>
   <br><br>
