<h1><b>Excel-Based Question Answering System using FAISS & Hugging Face LLM</b></h1>

<h2>📜 Overview</h2>

<h4>This project is a Retrieval-Augmented Generation (RAG) model that enables question answering over Excel data. It utilizes FAISS (Facebook AI Similarity Search) for efficient text retrieval and a Large Language Model (LLM) from Hugging Face for generating responses.</h4>

<h3>🚀 Features</h3>

<h4>✅ Query Excel Data Using Natural Language 📊✅ Fast Data Retrieval Using FAISS ⚡✅ Context-Aware Answers from LLM 🤖✅ No Hallucination - Answers Grounded in Provided Data ✅✅ Useful for Enterprise & Business Data Analysis</h4>


<h2>How It Works : </h2>

<h4>1️⃣ Load & Process Excel Data: Reads .xlsx files and converts them into text format.2️⃣ Create & Store Embeddings: Converts text chunks into numerical embeddings and stores them in FAISS.3️⃣ Set Up a Hugging Face LLM: Uses Mistral-7B-Instruct via the Hugging Face API.4️⃣ Retrieve & Answer Questions: Retrieves relevant text using FAISS and passes it to the LLM for response generation.</h4>
